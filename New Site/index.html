Introduction

On this Page you will find a puzzle game and a calculation tool. Please solve the puzzle first before using the tool. Read the rules below before pressing the button to start the puzzle.

Rules

  1. First enter a random number in each of the four fields
  2. Then press the submit button and have a look at the answer
  3. Try to find out how the four numbers are related to each other (which formula)
  4. If you get the answer, click on the "show answer" button
  5. if not, use different numbers and try again!
  6. Have Fun!
